Professional Overview

I am a full stack software developer. My specialization is backend development and data management. My primary skills are based in working with the .NET framework in VB and C# for Windows development, the JVM with Java and Scala for backend development, C/C++, and on the LAMP stack for web development, specifically focusing on PHP and MySQL.

I am the most skilled in using C# for Windows Form/WPF application development. I would also consider myself highly skilled in data system/API development in object oriented programming languages, specifically Java, C++, or C#. I have also worked fairly extensively with HTML5/CSS3 and Bootstrap in developing responsive web pages.

Aside from my technical skills, I also have Agile project management experience using the Atlassian tool suite and managing a team of 5-10 developers as part of Maverick Systems, the company I was part of starting in April 2018. I am currently the CEO and Lead Developer (Maverick Systems website).
